---

Most AssetArcade API endpoints are public and read-only.

## Public endpoints

These endpoints do not require authentication:

- search
- asset detail
- manifest
- context
- integration
- capabilities
- branches
- versions
- publisher profile

## Authenticated endpoint

`GET /v1/assets/{assetId}/entitlement` requires a Clerk bearer token.

## Bearer token format

Send the token in the `Authorization` header:

```http
Authorization: Bearer <token>
