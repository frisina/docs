# AssetArcade API

AssetArcade is an agent-native asset marketplace API for discovery, manifests, compatibility metadata, integration guidance, branches, entitlements, and publisher profiles.

## Base URL

`/api`

## Current version

`/v1/`

## Format

All responses are JSON unless otherwise noted.

## What you can do

- Search for assets by capability
- Inspect asset manifests, context, and integration metadata
- Discover capabilities and branch/version information
- Check ownership for authenticated users
- Look up publisher profiles
