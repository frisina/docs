AssetArcade is an agent-native asset marketplace API for discovering assets, inspecting structured metadata, and retrieving implementation guidance.

## What the API is for

The API is designed for both human developers and AI agents.

You can use it to:

- search for assets by capability
- inspect manifests, context, and integration guidance
- explore branches and version history
- check user entitlement for a specific asset
- look up publisher profiles

## Base path

All endpoints are served under:

`/api/v1`

## Response style

The API is read-focused and resource-oriented:

- search responses are lightweight
- detail endpoints are deeper and more structured
- manifest, context, and integration each have focused endpoints
- versioning is path-based

## Current surface

The current API includes:

- agent search
- asset detail
- manifest, context, and integration endpoints
- capabilities taxonomy
- branch listing and version history
- entitlement checks
- publisher profiles
